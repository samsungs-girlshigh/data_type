파이썬의 주요 데이터 타입

우리가 수집하는 정보의 종류에 따라 최적의 그릇을 준비해야 합니다.

str

문자열 (String)

일반 문자의 데이터입니다. 파이썬에서는 따옴표(' ' 또는 " ")로 감싸 문자임을 구분합니다.

💡 주로 사용되는 데이터 예시
name = "홍길동"
phone = "010-1234-5678"
int

정수 (Integer)

소수점이 전혀 포함되지 않은 정수형 숫자 데이터입니다. 양수, 음수, 0을 표현할 수 있습니다.

💡 주로 사용되는 데이터 예시
students_count = 28
attendance_no = 15
float

실수 (Floating-point)

소수점을 포함하고 있는 정밀 소수형 데이터입니다. 수학적 측정값 표현에 최적입니다.

💡 주로 사용되는 데이터 예시
vision_left = 1.2
fine_dust = 35.8
bool

참/거짓 (Boolean)

참(True) 또는 거짓(False)으로만 상태를 표기하는 자료형입니다.

💡 주로 사용되는 데이터 예시
is_submitted = True
is_checked = False
list

리스트 (List)

대괄호([ ]) 안에 여러 데이터를 쉼표로 연결하여 묶어두는 모음 구조입니다.

💡 주로 사용되는 데이터 예시
members = ["강은서", "김민기", "정은우"]
scores = [100, 95, 85, 70]
dict

딕셔너리 (Dictionary)

중괄호({ }) 안에 사전처럼 고유 열쇠 키(Key)와 값(Value)을 연결해 묶은 매핑 데이터형입니다.

💡 주로 사용되는 데이터 예시
user_profile = {"이름": "홍길동", "나이": 18, "학급": 1}
dictionary_dict = {"apple": "사과", "banana": "바나나"}
