파이썬의 주요 데이터 타입

우리가 수집하는 정보의 종류에 따라 최적의 그릇을 준비해야 합니다.

str

문자열 (String)

키보드로 치는 모든 일반적인 문자 데이터입니다. 파이썬에서는 따옴표(' ' 또는 " ")로 감싸서 문자임을 명시합니다.

💡 주로 사용되는 데이터 예시
name = "홍길동"
phone = "010-1234-5678"
int

정수 (Integer)

소수점이 전혀 포함되지 않은 깔끔하게 맞아떨어지는 숫자형 데이터입니다. 양수, 음수, 0을 모두 표현할 수 있습니다.

💡 주로 사용되는 데이터 예시
students_count = 28
attendance_no = 15
float

실수 (Floating-point)

소수점을 포함하고 있는 정밀한 소수 데이터입니다. 수학적 연산 및 정확한 상태 값을 나타낼 때 최적입니다.

💡 주로 사용되는 데이터 예시
vision_left = 1.2
fine_dust = 35.8
bool

참/거짓 (Boolean)

세상의 모든 상황을 오직 참(True) 또는 거짓(False)으로만 엄격하게 매핑하여 구분 짓는 가장 단순하면서 강력한 자료형입니다.

💡 주로 사용되는 데이터 예시
is_submitted = True
is_checked = False
list

리스트 (List)

대괄호([ ]) 안에 여러 개의 데이터를 쉼표로 연결하여 한 번에 가두어 관리하는 순서형 상자 데이터입니다.

💡 주로 사용되는 데이터 예시
members = ["강은서", "김민기", "정은우"]
scores = [100, 95, 85, 70]
dict

딕셔너리 (Dictionary)

중괄호({ }) 안에 사전처럼 고유 열쇠인 키(Key)와 그에 해당하는 값(Value)을 한 쌍으로 묶은 매핑 데이터형입니다.

💡 주로 사용되는 데이터 예시
user_profile = {"이름": "홍길동", "나이": 18, "학급": 1}
dictionary_dict = {"apple": "사과", "banana": "바나나"}
